# The Harlequin Paradox

## Strategic Absurdity in Negotiation

This repository contains the complete LaTeX manuscript, Python simulations, generated figures, and supporting material for **The Harlequin Paradox: Strategic Absurdity in Negotiation** by Don Merrow.

## Repository Contents

- `main.tex` — LaTeX source
- `main.pdf` — Compiled paper
- `figures/` — Generated figures
- `scripts/` — Simulation and plotting scripts
- `requirements.txt` — Python dependencies
- `Makefile` — Build automation

## Build

Generate figures:

```bash
make figures
```

Compile the paper:

```bash
pdflatex main.tex
pdflatex main.tex
```

Or build everything:

```bash
make
```

## Python Requirements

```bash
pip install -r requirements.txt
```

## Research Note

The mathematical models and simulations are exploratory. Simulation outcomes depend on the selected assumptions, parameter distributions, and random seed. They should not be interpreted as empirical measurements of real-world negotiation outcomes.
